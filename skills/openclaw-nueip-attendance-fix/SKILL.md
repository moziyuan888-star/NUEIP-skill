# OpenClaw NUEIP 補卡技能

依照以下固定流程執行，除非使用者明確要求變更。

## 1) 開站與登入判斷

1. 使用瀏覽器開啟：`https://cloud.nueip.com/attendance_record`。
2. 偵測是否需要登入。
3. 若需要登入且查無可用帳號，詢問並收集：
   - 姓名（帳號別名）
   - 公司代碼
   - 員工編號
   - 密碼
4. 支援多組帳號：
   - 「幫我補卡」：使用預設帳號。
   - 「幫 XXX 補卡」：使用姓名 `XXX` 對應帳號。
5. 若找不到對應帳號，先補問帳號資訊再登入。

## 2) 帳號保存規則

1. 將「姓名 → 公司代碼/員工編號/密碼」保存於本地可重用儲存。
2. 之後任務優先套用已儲存資料，避免重複詢問。
3. 若登入失敗，提示更新憑證並覆寫舊資料。
4. 不在回覆中輸出明文密碼。

## 3) 進入補卡資料頁

1. 進入頁面後點「新增」。
2. 顯示當月出勤紀錄後，依補卡月份切換快速查詢：
   - 補上個月：點「上個月」。
   - 補本月：點「本月」。
3. 將使用者口語日期（例如 `3/16`）轉為當年度 ISO 日期（例如 `2026-03-16`）。
4. 找到該日期列最左側紅色 `+` 按鈕並點擊。

## 4) 補卡表單填寫

1. **時段選擇**
   - 使用者說補上班卡：點「上班」左側小圓圈。
   - 使用者說補下班卡：點「下班」左側小圓圈。
2. **補卡時間（下拉式）**
   - 補上班卡預設時間：`09:30`。
   - 補下班卡預設時間：`18:30`。
   - 第一個下拉選「時」，第二個下拉選「分」。
3. **說明欄**
   - 固定輸入：`補卡`。

## 5) 補卡次數回報與提交

1. 在提交前讀取頁面上的「當月補卡累積次數」或剩餘可補次數資訊。
2. 將可補卡剩餘次數（或目前累積次數）先回報給使用者。
3. 點擊「確定」完成補卡提交。
4. 提交成功後回報：
   - 帳號別名（姓名）
   - 補卡日期
   - 補上班或補下班
   - 補卡時間
   - 當月補卡次數資訊

## 6) 例外處理

1. 找不到目標日期列或紅色 `+` 按鈕時，先回報目前畫面狀態並請使用者確認月份。
2. 若使用者未明確說上班卡或下班卡，先詢問再填。
3. 若使用者有指定時間，優先使用指定時間；未指定才使用預設時間。
